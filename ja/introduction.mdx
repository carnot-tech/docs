# Jinbaflow ドキュメント

## 概要

Jinbaflow は、大規模言語モデル（LLM）エージェント向けのワークフロー作成を支援するために設計されたプラットフォームです。ユーザーは自然言語で指示を入力でき、コパイロットエージェントが YAML マニフェストファイルを生成します。これらのマニフェストファイルには、ランナー環境でワークフローを実行するために必要なすべての情報が含まれています。

主な特徴:
- コパイロットエージェントによる自然言語でのワークフロー作成
- YAML ベースのマニフェストファイルによるワークフロー定義
- 統合された豊富なツール群（LLM、Python、Notion、Gmail など）の利用
- 複雑なワークフローパターンへの対応:
  - 順次および並列実行
  - 条件分岐
  - ループ処理と繰り返し
  - 動的な変数参照
  - テンプレートベースの値生成

このプラットフォームは、さまざまなツールやサービスを統合し、柔軟かつ強力にオーケストレーションできるため、自動化、データ処理、統合タスクに最適です。
